# 학원 출결 및 CRM 시스템 개발 프로젝트 인수인계 요약본 (Project Handover Summary)

이 문서는 대화방의 용량 한계로 인해 새로운 채팅창을 열어 작업을 이어갈 때, 새 AI 개발자에게 기존 작업 내용을 그대로 공유하여 100% 동일한 컨텍스트로 이어서 개발하기 위해 작성된 요약 파일입니다.

---

## 1. 프로젝트 개요 & 시스템 구조
본 프로젝트는 **원장용 학원 관리 및 CRM 웹앱(원장앱)**과 **태블릿용 학생 출결용 키오스크 웹앱(학생앱)**으로 구성된 통합 관리 시스템입니다. 데이터베이스로는 **구글 스프레드시트**를 실시간 양방향(Read/Write) 연동하여 사용합니다.

* **GitHub 레포지토리**: `https://github.com/readinyoohyun/academy-attendance.git`
* **Vercel 운영 서버 주소**: 
  * 원장앱: `https://academy-attendance-nu.vercel.app` (또는 각 학원별 복제 주소)
  * 학생앱: `https://[앱주소]/kiosk.html`
* **개발 기술 스택**: Vanilla HTML, JS (ES6+), Vanilla CSS (TailwindCSS 사용 안 함), 구글 Apps Script (GAS)

---

## 2. 핵심 구현 기능 목록

### ① 양방향 실시간 구글 시트 연동 (Google Sheet Integration)
* 구글 스프레드시트의 데이터를 웹앱과 실시간 양방향 동기화합니다.
* 스프레드시트 내의 Apps Script를 웹앱 API(JSON Web App)로 배포하여 API 요청을 송수신합니다.
* 60초 간격의 백그라운드 자동 동기화 기능을 포함합니다.

### ② 상용 솔루션 패키지화 & 화이트라벨링 (Commercial Whitelabeling)
* **🏫 동적 학원 브랜딩**: 학원명, 로고 이미지 URL을 마스터 설정에서 바꾸면 원장앱 타이틀, 키오스크 메인 로고, 알림 메시지 등이 실시간으로 자동 변경됩니다.
* **🔒 마스터 비밀번호 설정**: 비밀번호(공란 시 즉시 진입)를 설정하여 권한이 없는 일반 사용자가 마스터 설정 메뉴에 진입할 수 없도록 차단합니다.
* **📥/📤 JSON 백업 및 1초 복원**: 현재 학원의 모든 설정(학원명, 시트 ID, API 키 등)을 `.json` 파일로 저장하고, 다른 학원 배포 시 해당 파일을 업로드하여 1초 만에 초기 세팅을 복구할 수 있습니다.

### ③ 출결 알림 문자/알림톡 발송 (SMS/Alimtalk)
* **로컬 게이트웨이 모드 (공휴대폰 연동)**: 와이파이 전용 태블릿(유심 없음)의 CORS 우회를 고려하여, 로컬 IP SMS 게이트웨이 앱과 연동하여 등/하원 시 문자를 자동 발송합니다.
* **클라우드 API 모드 (솔라피 연동)**: 솔라피(Solapi) API Key, API Secret을 직접 등록하여 별도의 휴대폰 연동 없이 알림톡 및 문자를 즉시 발송합니다.

### ④ 고도화된 UI/UX 및 최적화
* **화면 떨림(움찔거림) 및 스크롤 튀기 버그 완벽 수정**:
  * 백그라운드 자동 동기화 시 데이터가 달라지지 않았다면 불필요한 화면 재렌더링을 완전히 차단합니다.
  * 표의 데이터를 갱신할 때도 스크롤 위치(상하좌우) 및 브라우저 뷰포트 위치를 메모리에 캐싱(박제)했다가 복구하여 화면 튐이 전혀 없습니다.
* **입력 제어**: 숫자(교재 레벨 1~10-4) 및 이벤트(완료, 보강 등) 입력 시 자유 타이핑 대신 드롭다운 선택상자로 제한하여 구글 시트 데이터의 정합성을 보장합니다.
* **줄바꿈 처리**: 한 명 검색 탭 등에서 텍스트가 길어져도 화면 밖으로 삐져나가지 않고 자동 줄바꿈(`white-space: pre-wrap;`)을 지원합니다.
* **템플릿 제외 필터**: 시트의 1, 2행(안내사항 및 타이틀 헤더)은 출석부 목록 파싱에서 안전하게 예외 필터링합니다.

## 3. 핵심 파일 구성 & 상세 역할

1. **`index.html`**: 원장앱의 전체 레이아웃 마크업. 대시보드 탭, CRM 탭, 구글 시트 시뮬레이터 탭, 마스터 패키지 설정창 마크업을 포함합니다.
2. **`app.js`**: 메인 어플리케이션 코디네이터. 초기 설정 로드, 로컬 스토리지 데이터 관리, 타 모듈 간 이벤트 조율 및 마스터 패키지 설정 이벤트를 담당합니다.
3. **`sheet-sim.js`**: 구글 시트 브라우저 시뮬레이터 렌더링 엔진. 드롭다운 제어, 셀 수정 시 실시간 양방향 전송, 스크롤 박제 및 복원 기능을 처리합니다.
4. **`api.js`**: 구글 Apps Script Web Web API와의 실시간 Fetch 통신 연동 모듈. 데이터 변경 감지 비교기(Compare Data Analyzer)가 탑재되어 있습니다.
5. **`dashboard-manager.js`**: 원장용 출결 현황판 렌더링. 요일별, 시간대별 타임 슬롯 및 등/하원 실시간 경과 시간 계산기 타이머가 내장되어 있습니다.
6. **`crm-manager.js`**: 개별 학생별 밀착 관리 카드 렌더링 (이름 검색, 다음 교재 안내 문자 미리보기 발송, 상담/분석/전송 이력 CRUD 및 결석/보강 일정 지정).
7. **`sms-manager.js`**: 원장앱에서 개별 SMS 발송 및 문자 설정 보드 제어.
8. **`kiosk.html`**: 학생용 등/하원 번호판 패드 키오스크. `academy_name` 및 `logo_url`을 감지하여 동적 화이트라벨 로고로 적용됩니다.
